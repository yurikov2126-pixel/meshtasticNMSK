# Security Policy
If you find a vulnerability or have a security question, please contact contact@Mesh-NMSK.ru  
Please do not create public issues before discussion.

# Политика безопасности
Если нашли уязвимость или есть вопрос по безопасности, напишите на contact@Mesh-NMSK.ru  
Просим не создавать публичных issue до обсуждения.
