### Магнитная CDEBYE TX868-XPL-100

Магнитная антенна TX868-XPL Не обязательно данного бренда, они бывают в продаже у разных продавцов, проверяйте соответствие частоте (868 или 433 mhz) и обращайте внимания на отзывы покупателей, продавцы иногда отправляют антенну не нужного диапазона.

Обязательно использование противовеса (металлический поверхности - автомобильной крыши, кондиционера, металлического отлива или  металлической банки)
Протестировано, соответствует диапазону (868mhz)

### Портативная 21см (штырь)

Средняя по размерам антенна для портативных нод, можно использовать так же в стационарном режиме
Данный продавец продает только лотами по 2шт, можно поискать у других продавцов
Завленная частота (868мгц) может не совпадать

Вероятно потребуется [доработка](https://i.imgur.com/v5XMaLQ.png) (удаление верхней части латунной трубки, можно сделать просто отломив руками)

Протестировано, соответствует диапазону после модификации (868mhz)



:::success
Стоимость: \~200p

Купить:

:::

Компактная всенаправленная антенна, с пластиной для монтажа 
Небольшие размеры и 1.5м кабель позволят удобно смонтировать ноду без потери в качестве сигнала

### Стеклопластиковая 50см

Большая всенаправленная антенна для постоянного монтажа, обязательно должна быть отнесена максимально далеко от стены/поверхности

Будет идеально работать при монтаже на мачте или выносе на крышу/конёк

### Магнитная Триада

Более дорогая магнитная антенна, испытана многолетней работой в сети Meshtastic, хороший вариант и для дома и для автомобиля 
Для данной антенны нужно использовать противовес (металл для опоры) 

Протестировано, соответствует диапазону (868mhz)
