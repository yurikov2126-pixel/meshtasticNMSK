# Дополнительные статические страницы

Эта директория предназначена для страниц, которые не входят в основное дерево документации.

- Файлы хранятся в `static-pages/` и обрабатываются плагином `@docusaurus/plugin-content-pages`.
- Разрешены Markdown и MDX. Можно подключать компоненты из `src/components`.
- В URL страница будет доступна по `slug`, указанному во front matter (по умолчанию совпадает с именем файла).

> Удалите или отредактируйте этот файл, когда появятся реальные материалы.
